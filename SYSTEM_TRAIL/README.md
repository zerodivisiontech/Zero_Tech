# System Trail

System Trail is a small experimental Python learning game.

The idea is simple: instead of reading lessons, you rebuild a broken digital world called **The System** by solving programming problems.

You wake up in the ruins of Data City with no memory. The knowledge that once powered the world was scattered when virus bandits destroyed the system. Your job is to recover the lost **Volumes of the System** and rebuild what was lost.

Each puzzle restores part of the world.

---
## The reason 

Finding a truly **free and interactive** way to learn Python was way harder than it should have been.

Most platforms lock the interesting parts behind subscriptions or paywalls.

This project is an attempt to build something different.

- No paywalls  
- No locked lessons  
- Just exploration and puzzles

---

## Current State

This is a **very early prototype**. still in progress

Right now the game includes:

- A small terminal-based story
- A code execution engine
- 5 scenes
- The first 3 scenes begin teaching basic Python concepts

The rest exist mainly to prove i can make the game system work and to provide a truly free way to learn.

---

## How to Run

From the project folder run:

python main.py

Type your Python code in the terminal.

When you're finished typing your solution, enter:

type End or END (tripped myself with that one)

The game will run your code and check the result.
